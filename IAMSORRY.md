## A Small Apology

If you happen to see commits with the message:

> Random rust code as usual!

...and this project isn't even written in Rust, that's on me.

At some point I set that as my default commit message for personal projects and, out of pure habit, occasionally forget to replace it before pushing. As a result, some commits in this repository may have a message that has absolutely nothing to do with the actual changes.

The code is intentional.

The commit messages are sometimes not.

I'll try to be more descriptive going forward.

Thanks for stopping by.
